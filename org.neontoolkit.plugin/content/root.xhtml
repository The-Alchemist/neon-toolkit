About the NeOn Project

NeOn is research project involving 14 European partners and co-funded by the European Commission's Sixth Framework Programme under grant number IST-2005-027595. NeOn started in March 2006 and has a duration of 4 years. Our aim is to advance the state of the art in using ontologies for large-scale semantic applications in the distributed organizations. Particularly, we aim at improving the capability to handle multiple networked ontologies that exist in a particular context, are created collaboratively, and might be highly dynamic and constantly evolving.

About the NeOn Toolkit

The NeOn toolkit is part of the reference implementation of the NeOn architecture. It is based on the Ontology Engineering Environment OntoStudio (c). In contains plug-ins for ontology management and visualization. The core features include:

  • Basic Modeling of OWL-2 ontologies
  • Browsing and visualizing
  • Import and Export in different formats
More NeOn Extensions

Get more plug-ins from the NeOn partners from the NeOn Toolkit Update Site. In the toolkit, just go to Help > Software Updates > Find and install.

Additionally, a large number of plug-ins have been developed by the NeOn consortium to support many tasks of the ontology life cycle, including

  • Project management for ontology projects
  • Diagnosing and repairing networked ontologies
  • Support for modularization and ontology design patterns
  • Support for reasoning and query processing

Commercial extensions of the toolkit

A number of commercial plug-ins are also available for the toolkit particularly adding means for supporting F-logic modeling, such as

  • Rule Support: Graphical/Textual editing, debugging
  • Mediation: Graphical Mapping Editor, life-interpretation of mappings
  • Database Integration: Database schema import, database-access
  • Queries: Query-Editor and persistent queries
