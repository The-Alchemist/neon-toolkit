About the NeOn Project

NeOn is research project involving 14 European partners and co-funded by the European Commission's Sixth Framework Programme under grant number IST-2005-027595. NeOn started in March 2006 and has a duration of 4 years. Our aim is to advance the state of the art in using ontologies for large-scale semantic applications in the distributed organizations. Particularly, we aim at improving the capability to handle multiple networked ontologies that exist in a particular context, are created collaboratively, and might be highly dynamic and constantly evolving.

About the NeOn Toolkit

The NeOn Toolkit is a state-of-the-art, open source multi-platform ontology engineering environment, which provides comprehensive support to the ontology engineering life-cycle.

More NeOn Extensions

Get more plug-ins from the NeOn partners via the NeOn Toolkit Update Site. In the toolkit, just go to Help > Software Updates > Find and install.

A large number of plug-ins has been developed by the NeOn partners, which cover a variety of ontology engineering activities. These include Annotation and Documentation, Human-Ontology Interaction, Modularization and Customization, Ontology Debugging, Ontology Dynamics, Ontology Evaluation, Ontology Matching, Ontology Specification, Reasoning and Inference, and Reuse.
