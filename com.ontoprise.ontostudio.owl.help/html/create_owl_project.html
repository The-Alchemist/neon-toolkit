Create an OWL Project

Ontologies are managed within projects. To create an ontology project, proceed as follows:
  1. Create an ontology project.
    Select File >> New >> Project. The New Project dialog opens.



  2. Select Ontology Development Project.
    Open the folder Ontology and select Ontology Development Project.



  3. Click Next.
    In the Project Name text box, type in a name.
  4. Select the ontology language.
    Select OWL as the ontology language of the project.



  5. Select a datamodel type.
  6. Click Next.
    Depending on your selected datamodel type, further settings might be required.
    If you select RAM, no further settings need to be changed.
  7. Click Finish.
    The ontology project will be created. To discard your changes, click Cancel.
