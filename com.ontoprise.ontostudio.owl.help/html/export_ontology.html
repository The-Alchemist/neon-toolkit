Export an Ontology

The Export dialog allows you to save an ontology into a different file (and possibly in a different format).

Caution – Note that exporting a file might cause a loss of information. For details, see the warning messages of the Export dialog.

To export an ontology, proceed as follows:

  1. Select an ontology.
    Select an ontology in the Ontology Navigator.
  2. Export an ontology.
    Right-click the ontology and select Export. The Export dialog opens.



  3. Select an export destination.
    Open the folder NeOn Toolkit (Ontologies) and select an export destination.
    You can choose between the following export methods:
    • Filesystem. Export an ontology to the filesystem.
    • WebDAV. Export an ontology via WebDAV (Web-based Distributed Authoring and Versioning),
      which is a set of extensions to the Hypertext Transfer Protocol (HTTP) that allows users to
      collaboratively edit and manage files on remote World Wide Web servers.



  4. Specify the ontology to be exported.
    Enter the following information:
    • Select Project. Select the project from which you want to export the ontology.
    • Select Ontology. Select the ontology you want to export.
    • To File. Select a file name and file type of the ontology to be exported.
      The following file types are available:
      • OXML – A proprietary XML format for data storage.
      • F-logic – A knowledge representation and ontology language.
      • RDF – A language for representing information about resources in the World Wide Web.
      • RDFS – The schema language for RDF.
      • n-triple – A serialisation format for RDF graphs.
      • Turtle – A serialisation format for RDF graphs.
      • OWL/RDF – The official OWL 1.0 W3C recommendation.
      • OWL/XML – An XML presentation for OWL.
      • OWL 1.1 – An experimental extension of the official OWL 1.0 W3C recommendation.



  5. Click Finish.
    The ontology will be exported. To discard your changes, click Cancel.
