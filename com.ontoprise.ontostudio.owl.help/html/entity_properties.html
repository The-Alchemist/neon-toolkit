Entity Properties

The Entity Properties panel is the main work area for viewing and editing classes, properties, and individuals. There are seven different Entity Properties panels for the different entity types relevant for OWL ontologies

When selecting one of the nodes in the Ontology Navigator, the appropriate Entity Properties View will display the details for this entity.

  • OWL Ontology
  • OWL Class
  • Object Property
  • Datatype Property
  • Annotation Property
  • Datatype
  • Individual
